Tic-Tac-Toe Online 🎮

A simple multiplayer Tic-Tac-Toe game built with HTML, CSS, JavaScript, and Firebase for real-time play.

## 🔗 Live Demo  
➡️ **[Play Now](https://fauzdaar.github.io/Testing/index)**


---

📌 Features

✔ Multiplayer support (Play with a distant friend in real-time)
✔ Firebase-powered game state (Syncs moves instantly)
✔ Turn indicator (Shows whose turn it is)
✔ Win detection (Stops game after a win)
✔ Game reset button

🚀 How to Play

1️⃣ Open the Live Demo
2️⃣ Share the link with a friend
3️⃣ The first player gets "X", the second gets "O"
4️⃣ Click on a cell to make your move
5️⃣ Wait for your friend’s turn (turns are enforced)
6️⃣ First to get three in a row wins! 🏆
